# v3.4 双向评分 — 重走 2006-2008 三年回测

## 重新回到当下：2009 开局视角

「理性人」拿着 v3.4（双向评分 + v3.3b 加仓三重门）从 2006.01 重新走了一遍，到 2008.12 末，**当前状态**：

| 指标 | v3.4 | v3.3b | Δ |
|---|---:|---:|---:|
| 2006 末资产 | 202.5 万 | 202.5 万 | 0 |
| 2007 末资产 | 391.3 万 | 391.3 万 | 0 |
| **2008 末资产** | **336.8 万** | **323.0 万** | **+13.8 万** |
| 3 年 CAGR | **49.90%** | 47.70% | +2.2pp |
| 最大回撤 | -17.84% | -15.82% | +2.0pp |
| 2008 全年回报 | -13.92% | -14.73% | +0.81pp |

**结论：v3.4 在 2006-2007 与 v3.3b 完全一致，唯一差异出现在 2008 年末——当政策实弹+估值极端+流动性极宽松三重确认时，v3.4 加仓到 80%（v3.3b 只到 75%）。**

### 关键观察

**1. 牛市顶部 (2007.08-2008.03)**：v3.4 和 v3.3b 行为完全一致
- 评分都从 12 →16 → 13 → 14 → 14 → 13
- 仓位都锁在 20-30%
- 双向评分**没有破坏**风险控制能力

**2. 熊市下跌 (2008.04-2008.10)**：v3.4 收到的"机会信号"被加仓三重门拦住
- 2008.06 评分 9 → 基础 50%，但**当月跌-22.7%** 触发了"本月跌>15%" 闸门 → 维持 20%
- 这正是 v3.3b 的精华：评分系统会被"市场极端波动"否决
- 这阻止了"过早抄底"的悲剧

**3. 政策实弹时刻 (2008.11-2008.12)**：v3.4 的真正发力点
- 2008.11 评分 -3 → 75%（v3.3b 也是 75%）：四万亿+央行降息
- **2008.12 评分 -6 → 80%（v3.3b 是 75%）**：美联储零利率+央行降息+中央经济工作会议"积极财政+宽松货币"
- 双向评分捕捉到「机会信号叠加」给出了更激进的仓位

### 反例：纯评分版本（无三重门）

如果完全相信 v3.4 评分，没有加仓三重门：
- 2008.06 评分 9 → 50% 仓位 → 当月跌 22.7%
- 2008.10 评分 0 → 75% 仓位 → 当月跌 25.9%
- **2008 末 260 万 / MDD -38%**

这就是为什么"机会信号"必须配合"市场行为闸门"——评分告诉你方向，三重门告诉你时机。

## v3.4 框架最终形态

```
评分系统 (v3.4 双向)
├── 风险信号 → 高分 → 减仓
└── 机会信号 → 低分 → 加仓基础

加仓三重门 (v3.3b 保留)
├── 闸门1: 本月跌 >15% → 暂缓
├── 闸门2: 首次降息 <60 天 → 暂缓
├── 闸门3: 近3月2次跌 >10% → 暂缓
└── 放行条件 (任一)
     ├── 评分≤9 + 本月+ + 央行实际降息
     ├── 评分≤6 + 近2月+ + 重磅刺激
     ├── PE<15 且 PB<2 (估值极端兜底)
     └── 评分≤-5 (v3.4 新增：深度机会信号)
```
