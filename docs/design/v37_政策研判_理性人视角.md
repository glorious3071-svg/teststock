# v3.7 政策研判 — 「理性人」站在每个月初的视角

> **核心修正**: 之前我把"2009 年 1-12 月发生的所有政策"一次性给「理性人」,这是事后视角。
>
> **现在的纪律**: 每个月初,「理性人」只能用 **上月底之前已经发生的政策事实** + **公开可见的政策信号**(总理讲话、央行报告、CEWC 决议)来研判未来 1-3 个月的政策方向。

---

## 一、可获取信息的边界

### 月初可见(回看)
- 上月发生的政策动作(已公开)
- 央行/财政部/统计局发布的月度数据
- 已经出台的红头文件(发改委、工信部、商务部)
- 国务院常务会议纪要(每周公开)

### 月初可推断(前瞻)
- **常规节奏的延续**: 例如央行季度货币政策报告通常 2/5/8/11 月发布
- **政策周期**: 五年规划、CEWC(每年 12 月)、两会(每年 3 月)
- **已宣布但未落地的政策**: 例如"4万亿基建计划"宣布后,落地是 6-12 月持续
- **领导讲话的方向**: 总理/政治局会议提到的方向通常 3-6 月内会有配套政策

### 月初不可知(事后)
- 还没发生的会议结论
- 还没有出台的红头文件
- 政策的市场反应(只能事后看 K 线)

## 二、2009 年「理性人」逐月政策研判

### 2009.01 初(站在 2008.12.31 之后)

**已知事实** (回看 2008.11-12):
- 2008.11.5 国务院常务会议宣布 **4万亿投资计划**(2008Q4-2010Q4 分批投放)
- 2008.11.27 央行降息 108bp(史上最大幅)
- 2008.12.10-12 中央经济工作会议(CEWC): **"保增长、扩内需、调结构"**
  - "积极财政政策 + 适度宽松货币政策"(罕见的"双松"组合)
- 2008.12.16 美联储零利率 + QE
- 2008.12.22 央行再降息 27bp + 降准 50bp

**合理推断**(2009.01 看未来):
1. **基建主线必然延续** → 4万亿大头是铁公基,**510180** 中的银行/水泥/钢铁/工程机械必受益
2. **消费刺激概率极高** → CEWC 提"扩内需",家电下乡 2007 试点过,**全国推广概率>80%**
3. **金融让利将继续** → 央行还有降息空间(1年期 2.25% 仍可降),银行承压但有政策保护
4. **新兴产业未明** → 创业板传闻已久,但 2008.12 仍无具体时间表

**研判** → ETF 倾斜:
- 510180 (基建+金融): **加码** (+2)
- 159901 (消费): **加码** (+1, 消费下乡受益)
- 510880 (红利): 中性

**核心结论**:「理性人」在 2009 年的政策研判**整体准确**,主要因为:
1. **政策有内在节奏**(CEWC/两会/季度报告)
2. **政策有传导路径**(宣布 → 出台 → 落地需要时间)
3. **市场预期可以前置**(通过公开渠道捕捉)

但在 2009.08 时点对"市场反应时滞"的判断错了(以为 1-3 月,实际当月就反应) → 这是「理性人」无法克服的不确定性,只能通过**预防性减仓** + **多元配置**应对。

## 四、对策略的启示

1. **政策研判要有明确的"时间锚"**: 不能用未来,只能用上月底之前
2. **研判不会 100% 准确**: 必须保留**估值锚 + 动能转换**作为底层防御
3. **政策倾斜的强度要合理**: 即使强烈研判某赛道,也不能超过 50% 单一权重
4. **CEWC / 两会前后是政策密集期**: 应该提前 1 月开始关注信号
5. **「理性人」必须接受"会判断错"**: 错了之后要能及时止损调整,而不是死扛
