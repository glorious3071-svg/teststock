# Guarded Strategy Weighted Allocation Report

生成时间：2026-07-12

## 目标

上一版逻辑是：

- 年度评分卡决定总权益仓位。
- Guarded Top-5 决定买哪些指数。
- 入选后权益仓位内部等权。

本次只优化第三层：入选后的 5 个指数如何分配权重。选股和总仓位不变。

## 比较方案

脚本：

```bash
python3 scripts/backtest_guarded_strategy_weighted_allocation.py
```

比较的分配方式包括：

- 等权。
- 按 guarded score 线性/平方加权。
- 按 6月动量 rank 加权。
- 按 12月动量 rank 加权。
- 按 6月+12月动量组合加权。
- 按 6月波动率反向加权。
- 固定 rank 权重网格，例如 30/25/20/15/10。

所有方案都只使用年初可见特征，不使用当年收益决定权重。

## 最优结果

最优分配方式：按 `momentum_6m_rank` 在 Top-5 内归一化加权。

也就是：

```text
权益桶内部权重_i = momentum_6m_rank_i / sum(momentum_6m_rank_top5)
组合实际权重_i = 年度评分卡权益仓位 * 权益桶内部权重_i
```

## 回测表现

| 框架 | 终值 | 累计收益 | 年化 | 最大回撤 |
| --- | ---: | ---: | ---: | ---: |
| 等权 Top-5 + 评分卡控仓 | 171.02万 | 71.02% | 11.33% | -16.52% |
| 6月动量加权 Top-5 + 评分卡控仓 | 171.89万 | 71.89% | 11.44% | -16.33% |

年度表现：

| 年份 | 标的桶收益 | 评分卡后组合收益 | 年末资金 |
| ---: | ---: | ---: | ---: |
| 2021 | 36.03% | 30.92% | 130.92万 |
| 2022 | -16.74% | -13.93% | 112.68万 |
| 2023 | -3.98% | -2.79% | 109.55万 |
| 2024 | 23.15% | 21.03% | 132.59万 |
| 2025 | 32.71% | 29.64% | 171.89万 |

## 解释

优化带来的提升很小，说明 Top-5 内部本来分数和动量都比较接近，强行做集中加权意义不大。

按 6月动量加权的好处是：

- 2023 年把亏损从等权的 `-3.04%` 改善到 `-2.79%`。
- 2024 年组合收益从 `20.77%` 提升到 `21.03%`。
- 最大回撤略有改善，从 `-16.52%` 到 `-16.33%`。

但收益改善只有 `+0.87pp` 累计收益，不能算质变。

## 建议

可以采用 6月动量轻微加权，但不建议做更激进的 rank 权重或分数集中加权。

实务上更稳的规则是：

- 总权益仓位：年度评分卡决定。
- 选股：Guarded Top-5。
- 入选后：按 6月动量 rank 轻微倾斜。
- 单只权益桶权重不额外设置高集中度，实际结果通常接近等权。
