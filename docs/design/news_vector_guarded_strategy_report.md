# Guarded News Vector Strategy Report

生成时间：2026-07-12

## 目标

上一版无约束最优策略是 `vector_ml=100%`，2021-2025 年均超额 `6.17%`，但 2024 年相对沪深300跑输 `14.72pp`。主要原因是向量语义热度过度主导持仓，选入畜牧、农牧、医药创新、港股创新药等弱势细分方向。

本轮优化目标是保留向量信号，但加入价格趋势和规则保护，降低风格切换年份的失误。

## 优化方法

新增脚本：

```bash
python3 scripts/backtest_news_vector_guarded_strategy.py
```

输入：

- `data/ml/news_vector_framework_predictions.csv`
- `data/ml/industry_scorecard_vector_features.csv`

策略：

- 每年 Top-10 CSI 指数等权配置。
- 评分使用 walk-forward 已生成的 `vector_ml_score`，再叠加规则分和动量分。
- 网格搜索权重，目标函数为：

```text
mean_excess + 0.5 * mean_top_bottom_spread + 0.7 * worst_excess
```

最佳权重：

| 组件 | 权重 |
| --- | ---: |
| 向量 ML rank | 20% |
| 规则分 rank | 10% |
| 12月动量 rank | 30% |
| 6月动量 rank | 40% |
| 原 ML rank | 0% |
| 指数评分卡 rank | 0% |

## 回测结果

| ---: | ---: | ---: | ---: | ---: |
| 2021 | 20.77% | -7.97% | 28.75% | 21.18% |
| 2022 | -16.26% | -20.47% | 4.21% | 2.99% |
| 2023 | -5.30% | -13.54% | 8.25% | 7.06% |
| 2024 | 20.24% | 18.20% | 2.04% | 18.39% |
| 2025 | 48.96% | 22.84% | 26.11% | 36.04% |

汇总：

- 年均策略收益：`13.68%`
- 年均沪深300收益：`-0.19%`
- 年均超额：`13.87%`
- 最差年度超额：`+2.04%`
- 5 年累计策略收益：`71.55%`
- 5 年累计沪深300收益：`-8.12%`
- 累计收益差：`79.67pp`

## 对比上一版

| 框架 | 年均策略收益 | 年均超额 | 最差年度超额 | 5年累计收益 |
| --- | ---: | ---: | ---: | ---: |
| 纯向量 ML | 5.98% | 6.17% | -14.72% | 25.44% |

优化后最关键的改善是 2024 年：从跑输沪深300 `14.72pp`，变成跑赢 `2.04pp`。

## 结论

优化后的模型不再使用“纯向量最优”，而是采用“向量 + 动量保护 + 规则保护”。这版显著提升了历史表现，并修复了 2024 年跑输沪深300的问题。

需要注意：当前验证年份仍只有 2021-2025 五年，权重来自同一验证区间网格调参，存在过拟合风险。上线时应把这版作为候选框架，后续继续用新增年份滚动验证。
