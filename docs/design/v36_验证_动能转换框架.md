# v3.6 动能转换框架 — 验证结果

> **设计原则（用户洞见）**:
> **"决定市场行为的永远是市场本身。外部信号都需要转换成内部的动能才能决定走势。"**

---

## 一、核心改造

| 框架 | 逻辑 |
|---|---|
| **v3.4** | 外部信号 → 直接评分 → 仓位（线性,顶部失灵）|
| **v3.5** | 外部信号 + 领先指标 → 直接评分（误报 2007.05）|
| **v3.6** | 外部信号 **× 4 维动能转换器** → 评分 + 仓位档调整（乘积,贴近现实）|

### 4 维动能转换器(`momentum_transformer`)

| 维度 | 放大条件(>1) | 吸收条件(<1) |
|---|---|---|
| **A 政策预期** | 宽松末期 + 首次收紧暗示 = 2.0 | 紧缩已 price-in(累计加息+持续) = 0.3 |
| **B 资金性质** | 信贷溢出(M1>25 + 大贷款) = 1.5 | 居民储蓄主导 = 0.7 |
| **C 牛市动能** | 主升浪<12月 = 1.5 | 主升浪>24月 = 0.5 |
| **D 估值-盈利** | PE 涨速 >> EPS = 2.0 | 同步赛跑 = 0.5 |

### 两层联动

1. **过热信号 × 动能转换器** — 重新加权过热扣分
2. **利多 price-in 反吐** — A=2.0 时回吐 80% 的负 base 分（"看似利多 + 隐藏顶部"）
3. **仓位档叠加** — A=2.0 + base ≤ -3 + 过热信号存在 → 额外减一档

## 三、4 年完整回测

| 指标 | v3.4(基线)| **v3.6** | 差额 |
|---|---:|---:|---:|
| 终值(2009.12)| 491.9 万 | **516.9 万** | **+25 万** |
| CAGR | 48.9% | **50.8%** | **+1.9pp** |
| 2007.05-07 仓位 | 50%-60% | **50%-60%**（同步）| 0 |
| 2007.10 顶部仓位 | 20% | 20%（同步）| 0 |
| 2009.07 仓位 | 80% | **60%** | -20pp |
| 2009.08 仓位 | 90% | **75%** | -15pp |

**核心成就**:
- ✅ **不误报 2007.05**(v3.5 致命缺陷已消除)
- ✅ **救住 2009.07-08**(避开 -24% 暴跌的更多敞口)
- ✅ **完全复用 v3.4 在 2006-2009H1 的路径**(无副作用)
- ✅ **净增 25 万,纯利得自精准识别"隐藏顶部"**

## 四、本质洞察

> **同一组外部信号(PE 高、新基爆款、两融激增)在 2007.05 和 2009.07 看起来一样,但市场会把它们转换成完全不同的动能:**
> - **2007.05**: 长牛市 + 储蓄长钱 + 紧缩已 price-in + 估值-盈利同步 → 信号被**吸收**,市场继续上涨
> - **2009.07**: 短反弹 + 信贷短钱 + 央行首次微调 + 估值严重透支 → 信号被**放大**,次月崩盘

> **v3.4 看温度计,v3.6 看免疫系统。**

## 五、待解 / 后续工作

1. **v3.6 在 2010-2026 的适应性** — 需要逐年验证（特别是 2015 / 2020 等结构差异较大的年份）
2. **数据字段补充** — M1、累计加息、首次微调暗示等字段需要后续年份的填充
3. **2007.10 真顶部的额外强化** — 当前 v3.4/v3.6 都靠"卖出三重门"识别,可以考虑把"美联储反转 + PE>50"也接入 4 维转换器
