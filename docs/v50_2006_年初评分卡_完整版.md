# 2006 年初评分卡 · V5.0 完整版

**计算时点**：2005-12-31（年初定方向，as_of_date 卡死，严防上帝视角）
**评分版本**：V5.0（21 字段全接入）
**底层数据源**：MariaDB / teststock（23 张表）

---

## 一、总结果

| 维度 | 结果 |
|---|---|
| **总分** | **-6** |
| **评级带** | **机会显著**（低估值 + 低利率 + 宽政策 + 情绪冷清） |
| **加仓三闸门** | **False**（未过） |
| **三闸门理由** | 无政策实弹（央行非宽松 + 中央会议非积极 + 基本面未触底反弹） |

### 含义解读

V5.0 设计是「评分定方向 / 三闸门控仓位」：

- **评分 -6 / 机会显著** → 估值便宜、利率低、政策不紧、情绪低迷，方向上偏机会
- **三闸门 False** → 没有政策实弹（无大规模降息/降准/放水），不允许立即重仓
- **2006 年初操作含义**：仓位维持中性偏低，等政策催化或市场情绪转向后再加仓

## 四、加仓三闸门检查

V5.0 规定：评分 ≤ -5（加仓档）时，需满足以下 3 条中至少 2 条才允许加仓到 ≥80%。

| 闸门 | 字段 | 数据值 | 通过 |
|---|---|---|---|
| 1. 央行口径已宽松 | `pboc_tone` == 'loose' | 'neutral'（稳健） | ✗ |
| 2. 中央经济会议积极 | `central_meeting_tone` == 'expansionary' | 'neutral'（稳中求进） | ✗ |
| 3. 基本面触底反弹 | `ppi_yoy_change=='turn_positive'` OR `pmi_resume_expansion=True` | flat / False | ✗ |

**通过 0/3，闸门 False**。结论：评分提示机会显著，但**无政策实弹**，不能立即加仓。

## 六、与历史回测对照

| 年份 | 评分 | 评级带 | 三闸门 |
|---|---|---|---|
| **2006** | **-6** | **机会显著** | **False（无政策实弹）** |

| 2008 | +10 | 高风险 | False |
