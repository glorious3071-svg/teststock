> 时点:**2010-12-31**
> 起点资产(2009末):**1,343,272**

## 一、2010 发生了什么——震荡市的典型样本

**典型震荡市**:一路跌到 7 月初(-30%),然后反弹到 11 月初(几乎收复全部),再回落到年末 -12.5%。**没有突破 2009 年末高点**——三年中没有牛市。

### 1.2 估值演变

| --- | ---:| ---:| ---:|
| 2009-12-31 | 27.81 | 28.49 | 3.35 |
| 2010-03-31 | 23.03 | 23.84 | 3.01 |

**估值大幅收缩**:PE-TTM 从 28.49 → 15.34,**几乎腰斩**。
**关键观察**:沪深300 -12.5%,但 PE-TTM -46%——意味着**盈利同比 +60%+**。整年挣的钱被估值杀回去了。

### 1.3 9 个行业全年表现——经典二八分化

| 主要消费 | -14.8% | +38.1% | **+17.7%** | -17.9% |
| 工业 | -21.4% | +32.8% | +4.4% | -22.8% |
| 电信 | -22.0% | +31.1% | +2.3% | -25.2% |

| 能源 | -37.5% | +42.9% | -10.6% | -39.9% |
| 公用事业 | -22.8% | +6.1% | -18.1% | -23.8% |
| **金融** | -30.1% | +4.5% | **-26.9%** | -30.7% |
| 沪深300 | — | — | **-12.5%** | -29.7% |

**两组明显分化**:
- **赢家**:信息技术/医药/消费 全年 +18~41%(增长股 + 抗通胀)
- **输家**:金融 -27%、公用 -18%、能源 -11%(政策最受压制)

**理性人 2009 年末退出金融、加入电信公用是否正确?**
- ✅ **退出金融正确**:金融 -26.9% 全年最差
- ✅ **加入信息/医药/消费正确**:全年 +18~41% 全年最强
- ⚠️ **加入公用事业并不好**:全年 -18.1% 也是输家
- ⚠️ **电信仅 +2.3%**:H2 反弹但 H1 拖累

## 六、为什么仓位 60% 也能跑赢满仓 +20 点?

### 6.1 三个 alpha 来源

| 来源 | 贡献(估算) |
| --- | ---:|
| **行业选择**:退出金融,保留医药/消费/信息 | +12% |
| **季度再平衡**:H1 末低位补仓信息/电信 | +5% |
| **持有 40% 类现金**:类现金 +3.5% vs 股市 -12.5% | +6% |
| 合计超额 | ≈ +23% |

(实际超额约 +20%,余下差异来自买卖时点和摩擦成本)

### 6.2 关键决策的事后检验

**对了的决策**:

| 决策 | 验证 |
| --- | --- |
| 2009.12 决定降到 60% | ✅ 沪深300 -12.5%,逃过 -30% 回撤 |
| 退出金融 | ✅ 金融全年 -26.9% 最差 |
| 退出工业 | ✅ 工业全年仅 +4.4% |
| 保留信息技术 | ✅ +40.6% 全年最强 |
| 保留医药 | ✅ +23.8% 第二强 |
| 保留主要消费 | ✅ +17.7% 第三强 |
| 季度再平衡机制 | ✅ 7 月低点自动加仓 |

**错了的决策**:

| 决策 | 验证 |
| --- | --- |
| 加入公用事业 | ❌ 全年 -18.1%,拖累 |
| 加入电信 | ⚠️ 全年 +2.3%,中规中矩 |
| 减仓节奏太均匀 | ⚠️ 如果 1.4 一次性减完会少 -3% 回撤 |

**改进空间**:**避极端框架不要追求"防御性"**,公用事业不属于增长股,在通胀+收紧环境里也并非真正避险。

## 七、心理观察:震荡市最难

### 7.1 三次"考验"

**第一次:6-7 月信仰危机**
- 资产从 1.35M 跌到 1.18M,-12%
- 此时已经减仓到 60%,但还是跌
- 媒体叙事:"二次探底"、"双底"、"上证还会破 1664"
- **诱惑**:"是不是该再降到 40%?"

**第二次:8-10 月反弹诱惑**
- 信息技术 7-10 月 +35%
- **诱惑**:"应该一把切到信息技术上!"
- 不该听:已经涨了,再追就是真追高

**第三次:11 月顶部恐惧**
- 11-8 沪深300 几乎收复全年,媒体叙事翻转
- "牛市重启"、"2011 会冲 4500"
- **诱惑**:"应该把现金加回到股票!"
- 不该听:估值短暂回到 21x,但通胀和加息周期才刚开始

### 7.2 避极端框架的 2010 表现

**指标是 39%,处于"中性偏机会"区——对应仓位 55-65%**。
理性人 60% 仓位 **正好在框架建议区间内**。

**全年仓位不需要再调整**——这是"避极端"的优势:**只在状态显著变化时操作**。

## 八、2010 年末决策:进入 2011 怎么办?

### 8.1 状态判断

| 维度 | 状态 |
| --- | --- |
| **估值**:PE-TTM 15.34,5 年中位附近 | 中性 |
| **政策**:加息周期已经开始(加 2 次),2011 还会继续 | 偏紧 |
| **通胀**:CPI 5.1%,远超目标 | 警戒 |
| **外部**:欧债扩散,美国 QE2 启动 | 不确定 |
| **总指标** | **39% 中性偏机会** |

### 8.2 不需要大调整

**保持 60% 仓位**,但行业结构小调:

| 当前 | 2011 计划 | 原因 |
| --- | --- | --- |
| 主消 12% ✅ | 主消 12% | 抗通胀,业绩稳健 |
| 医药 12% ✅ | 医药 12% | 抗通胀,业绩好 |
| 信息 12% ✅ | 信息 12% | 估值还不算高 |
| 电信 12% | **电信 6%** | 表现一般,减半 |
| **公用 12%** | **公用 0%(退出)** | 通胀环境最受压制 |
| (空) | **金融 6%(回补)** | 估值已经杀到极低 |
| (空) | **能源 6%(新进)** | 抗通胀,估值便宜 |

新组合:主消12 + 医药12 + 信息12 + 电信6 + 金融6 + 能源6 = **54%**
(略低于 60%,因为 CPI 仍在高位)

### 8.3 触发再加仓的信号

| --- | --- |

| 沪深300 PE-TTM < 12 | 仓位 +10% |
| 国务院"稳增长"基调出现 | 仓位 +5% |

## 十、给后续年份的认知积累

### 10.1 2010 验证了什么

1. **季度再平衡是真 alpha**——震荡市里机械执行优于人为判断
2. **"避险"行业不一定真避险**——通胀+收紧环境里公用事业最受伤
3. **不需要追逐"最强行业"**——均衡5个里有 3 个跑赢沪深300就够
4. **状态稳定时不要瞎动**——39% 中性区,60% 仓位坚持一整年
5. **政策周期是可观察的**——加息+收紧已经从 2009.12 信号读到,2010 只是验证

### 10.2 进入 2011 要关注的信号

| 信号 | 关注理由 |
| --- | --- |
| **CPI 拐点** | 通胀见顶决定加息周期是否结束 |
| **货币政策"中性"措辞** | 从"稳健"到"中性"是宽松前兆 |
| **欧债是否传染到银行系统** | 雷曼级风险预警 |
| **小盘股估值** | 2010 创业板高估,2011 可能崩 |
| **房地产调控加码或松动** | 重要资产类别走向 |

## 十一、一句话总结

> **2010 年理性人做对的不是"减仓",而是"在 60% 仓位上坚定不动"。**
>
> 全年最难的不是 1 月的减仓决策(那是 2009.12 就做好的),
> 而是 6-7 月时熬住继续杀的痛苦,
> 是 11 月反弹时压住"加回去"的冲动。
>
> **避极端框架的真正价值是"少操作":全年只有 3 次主动操作(1.15/2.15/3.15),其他都是机械再平衡**。
>
> 三年综合 +44.24%,**相对满仓策略多 86 个百分点**——
> **不是因为做了什么聪明事,是因为没做愚蠢事**。
