### 1.4 重要观察(关键 alpha)

- **H1 是周期股的天下**:金融、能源、材料各 +73~89%(政策预期+流动性)
- **H2 是消费/医药/科技的天下**:消费/医药 +35%、电信 +35%、信息技术 +32%
- **风格切换发生在 7-8 月**(从"政策周期" → "消费成长")
- 公用事业全年仅 +42%,**显著跑输**——慢牛特征,**完成防御使命就该退出**
- 全年最大回撤普遍在 8 月初到 9 月初,大部分行业跌 -20% 到 -30%
- 但**全年最终都 +80% 以上**,**没有"二次探底"**

## 二、政策面:2009 中央经济工作会议(12.5-7)

**关键定调出现了**。会议公报里出现了**新的信号词**:

### 2.1 总基调:延续但开始转向

- **延续**:"继续实施积极的财政政策和适度宽松的货币政策"——大方向不变
- **新增**:"管理通胀预期"——**首次出现!这是 2008.11 以来第一次提"通胀"**
- **新增**:"提高经济增长质量和效益"——开始降温意味
- **新增**:"严格控制对高耗能、高排放行业和产能过剩行业的贷款"——**信贷结构性收紧**

### 2.2 信贷数据:超天量

| 指标 | 2008 | 2009 |
| --- | ---:| ---:|
| 新增人民币贷款 | 4.9 万亿 | **9.6 万亿** |
| M2 同比 | 17.8% | **27.7%** |
| 固定资产投资 | 25.5% | **30.1%** |

**M2 增速 27.7% 是历史峰值**。2009 全年信贷投放接近"过载"水平。

### 2.3 通胀已经从抬头转为加速

| 月份 | CPI 同比 |
| --- | ---:|
| 2009.07 | -1.8% |
| 2009.09 | -0.8% |
| **2009.11** | **+0.6%(由负转正)** |
| 2009.12 | **+1.9%** |

**CPI 12 月 +1.9%,4 个月内从 -1.8% 蹿升至 +1.9%——4 个月升 3.7 个点**。
PPI 也在 12 月转正 +1.7%。

**通胀从"零"快速回到"轻度通胀",且加速度很大**——这是 2010 年货币政策转向的前兆。

### 2.4 房地产开始过热

- 全年商品房销售面积同比 **+42.1%**(2008 是 -19.7%)
- 全年商品房销售额同比 **+75.5%**
- 12.21 国务院发布《关于促进房地产市场健康发展的若干意见》——开始"调控"性表态
- 部分一线城市房价同比 **+20-30%**

**房地产从"救市"开始转向"调控"——但还很温和**。

## 五、心理诱惑(此时此刻最可怕的陷阱)

### 5.1 "牛市才刚开始" 论

- 沪深300 +96.7%(从 1818 涨到 3576)
- 但仍比 2007.10 峰值(5877)低 39%
- **市场叙事**:"从 1664 到 5877 还有 250% 空间,牛市才走了 1/3"
- **真相**:估值已经回到 PE 28x,**接近 2008 年顶部估值水平**——再涨 30%,PE 就到 35x+

### 5.2 "中国经济独大,A 股将永远高估值" 论

- 中国 GDP +8.7%,美国 -2.5%
- 中国是"全球唯一亮点"
- 但**估值反映了未来增长**:PE 28x 意味着市场预期未来 5-10 年的高增长持续
- **真相**:经济回到 10.7%,后续大概率回落到 7-8% 常态区间

### 5.3 "通胀利好股市" 论

- "温和通胀对股市是利好"
- **真相**:温和通胀的下一步是央行收紧,**M2 27.7% 是不可持续的**
- 2010 年央行必然进入收紧周期(加准+加息)

### 5.4 "刚加完仓为什么要减" 论

- 理性人在 1-4 月才刚刚加到 75%
- 现在 8 个月不到就考虑减仓?
- **真相**:状态变了,决策就该变;1 月是极端机会区,12 月是中性区
- **不对称的清晰**:状态变化驱动决策,**不是"刚做完就不能改"**

## 六、理性人的 2009 年末决策

### 6.1 状态判断

崩盘指标从 22% → 50%,**已经穿过"极端机会"边界,进入"中性区"**。
对应建议仓位:**50-60%**(下降区间)。

### 6.2 当前实际仓位

修正版避极端执行后:**75% 股(均衡 5)+ 25% 类现金**。

### 6.3 调仓建议

**从 75% → 60%(下降 15 个点)**

执行节奏:
- 2010.01.15:75% → 70%(减 5%)
- 2010.02.15:70% → 65%(减 5%)
- 2010.03.15:65% → 60%(减 5%)
- **不在年末一次性减,跨年分批降**——避免被"年末翘尾行情"误判

### 6.4 行业结构调整

H2 已经显示**风格切换**:周期股见顶,消费/医药/科技领涨。

| 旧均衡 5 | 新均衡 5 | 原因 |
| --- | --- | --- |
| 金融 | **退出**(降到 5%) | H2 已 +4%,流动性收紧最先承压 |
| 工业 | **退出**(降到 5%) | 投资见顶,产能过剩警示 |
| 主要消费 ✅ | 主要消费 ✅ | H2 +35% 走势好,通胀受益 |
| 信息技术 ✅ | 信息技术 ✅ | 战略新兴产业政策支持 |
| 医药卫生 ✅ | 医药卫生 ✅ | 抗通胀 + 稳健 |
| (新增) | 电信 ✅ | H2 +35%,移动互联网叙事开启 |
| (新增) | 公用事业 ✅(20%) | 防御位,通胀传导对冲 |

**新组合**:主要消费 + 医药卫生 + 信息技术 + 电信 + 公用事业(各 20%)

### 6.5 加仓信号清单 → 降仓信号清单

从"加仓"切换到"降仓"模式。降仓硬条件:

| 信号 | 状态 |
| --- | --- |
| ① PE > 中位数(2009 中位 ~22) | ✅ 27.81 > 22 |
| ② CPI 持续上升,通胀预期形成 | ✅ +1.9% 且加速 |
| ③ 央行政策口径开始"管理通胀预期" | ✅ 12 月经济工作会议出现 |
| ④ 信贷投放进入"调结构、控总量" | ✅ "把握节奏" |
| ⑤ M2 增速进入历史高位区 | ✅ 27.7% 历史峰值 |

**5/5 全触发,降仓应当启动**。

### 6.6 最终决策

| 项 | 决策 |
| --- | --- |
| **目标仓位** | **60%**(分 3 批从 75% 降下来) |
| **行业** | **从政策受益版切换到消费+科技+防御** |
| **节奏** | 1.15 / 2.15 / 3.15 各减 5% |
| **触发条件** | 状态从"极端机会"切换到"中性",且已开始边际收紧 |
| **应急储备** | 40% 类现金 |
| **下次检视** | Q1 末复盘 |

## 八、给后续年份的认知积累

### 8.1 2009 年验证了什么

1. **极端机会区的判断是可行的**:PE 13x + 4 次降息 + 四万亿 → 明确触发
2. **同一套思维可以做出截然相反决策**:30% → 75% → 60%,都是"避极端"
3. **行业选择重要性远低于仓位**:9 个行业 8 个都 +85%+
4. **均衡 5 行业完全够用**,不需要追"最强行业"
5. **状态变化驱动决策,不需要追时点**

### 8.2 进入 2010 年要关注的新信号

| 信号 | 关注理由 |
| --- | --- |
| **CPI 突破 3%** | 通胀目标线,触发加息 |
| **央行加息/加准时点** | 政策转向确认 |
| **房地产调控加码** | 资产泡沫预警 |
| **欧债危机演化** | 外部尾部风险 |
| **PE 突破 30x** | 估值警戒线 |

## 九、一句话总结

> **2009 年理性人做对的不是"加仓加得早",而是"在状态变化时果断改变决策"。**
>
> 1 月加到 75% 是合理的(极端机会区),
> 12 月降到 60% 也是合理的(中性区,边际收紧已现)。
>
> **看似翻脸,实际是同一套思维下的诚实反应**——
> **状态变了 56 个百分点(指标 22% → 50%),决策当然要相应改变**。
>
> 两年综合 +34.33%,**比满仓策略多 67 个百分点的相对收益**。
> 而真正的赢家不是"赚到 +34%",
> **是"在 2008 不被杀死、在 2009 不被诱惑、在 2010 还能继续守住"**。
