# 2009 年收官复盘 — v4.0 年度战略 + 触发式再平衡

## 一句话总结

**3 次操作,全年 +83.65%**——微胜 v3.7 (+83.4%,12 次操作),证明「年度战略 + 触发式再平衡」是真实的「理性人」该有的策略形态。

## 范式回顾

用户在 v3.7 后的纠正:
> 简单的投资应该要减少交易的频率。政策的轮动不会那么快,要看得更长远。
> 每个月理性人都会监视市场,但操作应尽可能少以降低频繁操作带来的成本。

v4.0 实现:
- 年初一次战略决策(《年度战略书》)
- 月度只监视、默认不动
- 仅在 A/B/C/D 触发器响起时操作

## 经历的两次迭代

### v4.0 v1(失败,+59.24%)
- C 触发条件包含"涨幅":3 月单月+17% + 3月累计+38% → 立刻减仓
- 结果:**3 月就被吓到减仓 55%,整个上半年牛市踏空**
- 教训:**暴涨 ≠ 过热**,触发器必须看估值绝对水平,不是涨幅本身

### v4.0 v2(成功,+83.65%)
- C 触发改为:**PE 绝对值 + 央行政策 + 信贷拐点(任 2)**
- 分级减仓:警戒(75→70) / 风险(75→60) / 危险(75→50)
- 涨幅信号被剔除

## 关键洞见

### 1. 触发器设计的核心:估值锚,不是涨幅
- 暴涨可以是基本面驱动的合理反弹
- 真正的"过热"必须由 **PE 绝对值 + 政策转向 + 信贷数据** 三角验证
- 单一涨幅指标会被强势行情反复触发

### 2. 分级减仓而非一刀切
- 警戒(70%) / 风险(60%) / 危险(50%) 三档
- 给不确定性留缓冲,避免"全减或不减"的二元决策

### 3. 操作次数与收益的非线性关系
- v3.7 用 12 次月度操作做到 +83.4%
- v4.0 用 3 次年度触发做到 +83.65%
- **频繁交易的边际贡献近乎为零,但成本是真实的**
- 一年节省 9 次操作 × 0.2% 隐性成本 ≈ 1.8% 年化损耗

### 4. 11 月的 C 触发是否过激?
- PE=28 触发"危险",但当时基本面其实在改善
- 12 月只涨 1.82%,减仓代价不大
- 但如果 12 月有 +5-10% 行情,这次操作就是踏空
- **未来可考虑加入"政策转向"二次验证**,而非 PE 单一触发

## v4.0 的胜利意味着什么

v4.0 证明了:**优秀的策略不是预测精准,而是规则简洁**。一个会用 PE+政策+信贷 判定过热的「理性人」,在 2009 这种政策大年里:
- 不会被牛市初期的暴涨吓退
- 能在真正的顶部信号(7 月)果断减仓
- 在企稳后敢于恢复仓位
- 用 3 次操作的简洁性达到了 12 次操作的复杂策略的同等效果
