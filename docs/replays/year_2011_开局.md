# 2011 年初:理性人看到了什么,得出什么结论

> 时点:**2011-01-01**
> 视角约束:严格使用 2010-12-31 之前可见的数据
> 起点资产(2010末):**1,442,376**
> 起始仓位(2010末实际): 60% 股(主消/医药/信息/电信/公用 各12%) + 40% 类现金

---

## 一、3 年走完一轮——理性人看到的轨迹

### 1.1 沪深300 历史回放

| 时点 | 收盘 | PE-TTM |
| --- | ---:| ---:|
| 2007-10-16(历史顶) | 5877 | **47.79** |
| 2008-10-28(熊市底) | 1664 | 12.51 |
| 2008-12-31 | 1818 | 12.93 |
| 2009-08-04(09 高点) | 3787 | **31.91** |
| 2009-12-31 | 3576 | 28.49 |
| 2010-07-05(10 内低) | 2513 | 14.96 |
| 2010-11-08(10 反弹峰) | 3549 | 17.09 |
| **2010-12-31** | **3128** | **15.34** |

**理性人能看到的事实**:
- 沪深300 已经从 5877 跌到 3128,**3 年累计 -47%**
- 从 2008.10 底部 1664 反弹的部分已经回吐了 1/3
- **PE-TTM 15.34x,接近 5 年中位数,估值不贵但也不极便宜**
- 没有出现 2008.10 那种"极端机会"(PE 12.5x + 政策全面救市)

### 1.2 行业 3 年累计的真实情况

| 行业 | 2008 | 2009 | 2010 | **3 年累计** |
| --- | ---:| ---:| ---:| ---:|
| **医药卫生** | -44.7% | +97.9% | +23.8% | **+35.6%** |
| **信息技术** | -64.6% | +115.6% | +40.6% | **+7.3%** |
| 主要消费 | -56.7% | +95.9% | +17.7% | -0.2% |
| 电信 | -55.2% | +112.6% | +2.3% | -2.5% |
| 工业 | -66.3% | +86.1% | +4.4% | -34.5% |
| 材料 | -70.6% | +121.9% | -0.5% | -35.1% |
| 能源 | -68.8% | +131.2% | -10.6% | -35.5% |
| **公用事业** | -55.9% | +46.3% | -18.1% | **-47.1%** |
| **金融** | -66.8% | +104.8% | -26.9% | **-50.3%** |

**关键观察**:
- **只有医药、信息、主消、电信跑赢/打平起点**——这是真正的"业绩驱动"行业
- **金融、公用 3 年腰斩**——周期性 + 政策压制
- **没有任何一个行业回到 2007.10 起点**(3 年都跑不平 -47%)
- 行业表现的**前后一致性**:**业绩稳定的行业(医药/消费/信息)穿越周期**

## 二、宏观环境:四个相互冲突的力量

### 2.1 通胀:依然在高位

| 月份 | CPI |
| --- | ---:|
| 2010-07 | +3.3%(突破目标) |
| 2010-10 | +4.4% |
| **2010-11** | **+5.1%(28 月新高)** |
| 2010-12 | **+4.6%**(略回落) |

**问题**:CPI 是真见顶还是假回落?
- PPI 也在 5.9%(12 月),滞后传导仍在路上
- 国际大宗商品(原油 / 铁矿 / 棉花)2010 年大涨,2011 还有传导
- **理性人判断:CPI 大概率 2011.H1 还会冲到 6%+,然后才回落**

### 2.2 货币政策:紧缩刚刚开始

| 指标 | 2010 末状态 | 2011 预期 |
| --- | --- | --- |
| 1 年期存款利率 | 2.75% | 大概率 →3.5%+(还有 2-3 次加息) |
| 存款准备金率 | 18.5%(大行) | 可能 → 20%+ |
| M2 增速 | 19.7% | 目标 16% 附近——继续收缩 |
| 中央经济工作会议表述 | "把稳定价格总水平放在更加突出的位置" | 通胀是 2011 头号任务 |

**关键判断**:**2011 是紧缩主战场**,而不是 2010。2010 只是"开始转向",2011 是"硬约束"。

### 2.3 经济增长:从过热往常态回归

| 季度 | GDP |
| --- | ---:|
| 2010 Q1 | +12.1% |
| 2010 Q2 | +10.3% |
| 2010 Q3 | +9.6% |
| 2010 Q4 | +9.8% |

**趋势**:已经在降温,2011 预期 9% 左右。"软着陆"是政策目标。

**风险点**:**滞胀风险**——紧缩 + 增长放缓 + 通胀高位,这是最难的市场环境。

### 2.4 地产调控:不会松

- 2010.04 国十条 / 2010.09 五项 / 2010.12 增加保障房 1000 万套
- 限购扩到 16 城,2011 大概率扩到 30+ 城
- "房地产是支柱产业"和"房地产价格过高"的矛盾
- **理性人判断**:2011 地产投资可能继续下行,但不会断崖

### 2.5 欧债:不确定性的最大来源

- 已发生:希腊(2010.05)、爱尔兰(2010.11)
- **市场担忧**:葡萄牙、西班牙、意大利
- 美国 QE2 已经启动(11 月 6000 亿)
- **理性人判断**:欧债的尾部风险高于 2010,但传染到中国不直接,主要通过出口

## 三、估值:不极端,不极便宜

### 3.1 沪深300 的位置

| 维度 | 2010-12-31 | 历史百分位(用 2005 起的数据估算) |
| --- | --- | --- |
| PE-TTM | 15.34 | ~30 百分位 |
| PB | 2.31 | ~30 百分位 |
| 相对 2007.10 顶 | -47% | — |
| 相对 2008.10 底 | +88% | — |

**判断**:估值"中性偏便宜",**但远未到"极端机会"**。
- 极端机会区(PE 12-13x):需要再跌 15-20%
- 中性区(PE 15-17x):现在的位置

### 3.2 大小盘的极端分化

| 板块 | 2010-12-31 PE-TTM |
| --- | ---:|
| 沪深300 | **15.34** |
| 中证500 | ~30 |
| 创业板指 | **~78** |

**这是一个极端不均衡的市场**:
- 大盘股已经便宜(银行 PB < 1.5,部分接近 1.0)
- 小盘股已经极贵(创业板 PE 78x,接近 2007 沪深 300 顶部)

**理性人结论**:**未来回归** = 大盘补涨 + 小盘补跌 或 大盘持平 + 小盘暴跌

## 四、系统性崩盘指标(2011-01-01 视角)

| 维度 | 2010-12 | 备注 |
| --- | ---:| --- |
| **估值**(沪深300 维度) | 3/18 | PE 15.34,中性偏低 |
| **市场结构**(创业板大热,大蓝筹被嫌弃) | 9/18 | 散户已被中小板捕获 |
| **流动性政策** | 10/18 | 加息周期刚启动,2011 是主战场 |

**注**:估值维度看沪深300 不算高,但**结构性维度看,小盘股极端高估**,这是个"被隐藏的极端"。

## 五、心理诱惑:2011 年最危险的 3 个故事

### 5.1 "创业板 + 中小板代表中国未来"

- 2010 创业板平均 +30%,中小板指 +21%
- 主流叙事:"新兴产业、消费升级、医药创新——小盘股是未来"
- **诱惑**:把仓位从沪深300 一级行业切换到中小板/创业板
- **真相**:PE 78x 的小盘股 = 2007 沪深300 顶部估值水平,**这是次生泡沫**

### 5.2 "PE 15x 已经很便宜,可以加仓"

- 沪深300 PE-TTM 15.34,看起来便宜
- **诱惑**:把仓位从 60% 加回 70%
- **真相**:
  - 2008.10 底部 PE-TTM 12.51——现在还高 23%
  - 加息周期才刚开始,通胀还在高位
  - **"便宜"是相对的,在紧缩 + 高通胀环境下,15x 也可能继续跌**

### 5.3 "通胀就要见顶了"

- 12 月 CPI +4.6% 比 11 月 +5.1% 已经回落
- **诱惑**:"通胀压力解除,可以加大蓝筹"
- **真相**:
  - 12 月翘尾因素改善,1-2 月会因为春节又冲高
  - PPI 还在 5.9%,传导未完成
  - 食品价格仍在上涨

## 六、"避极端"框架下的 2011 年初决策

### 6.1 状态判断

指标 33/72 = **46%,中性偏紧**。

参考映射表:

| 指标 | 区间 | 建议股仓位 |
| --- | --- | --- |
| ≥75% | 极端危险 | 20-30% |
| 60-75% | 危险 | 35-45% |
| **40-60%** | **中性** | **50-60%** |
| 25-40% | 机会 | 60-70% |
| <25% | 极端机会 | 70-85% |

对应股仓位:**50-60% 区间**(下沿)。

### 6.2 当前仓位回顾

2010 末实际:**60% 股 + 40% 现金**——已经在区间下沿,**不需要大幅调整**。

### 6.3 但有一个结构性问题

2010 验证了:**"防御性"行业(公用事业)不一定真避险**——通胀环境里反而最受压制。

需要重新审视**为什么持有公用事业**:
- 假设:防御 + 现金流稳定
- 现实:通胀环境里电价滞涨成本上涨,业绩反而被吞噬
- 结论:**公用事业不适合通胀+紧缩环境**

### 6.4 行业调整建议

**保留**(都是 3 年穿越周期的赢家):
- 主消 12%(食品饮料抗通胀)
- 医药 12%(刚性需求 + 政策医改)
- 信息 12%(估值不算太高,产业政策支持)

**调整**:

| 旧 | 新 | 原因 |
| --- | --- | --- |
| 电信 12% | **电信 6%** | 2010 仅 +2.3%,移动互联网叙事未兑现,减半 |
| **公用事业 12%** | **退出(0%)** | 通胀环境最不利 |
| (空) | **金融 6%** | PB 已经杀到 1.5 下,但加息周期对银行短期利好(息差扩大);保留小仓位试探 |
| (空) | **能源 6%** | 抗通胀属性,估值便宜(3 年 -36%);通胀传导利好 |

新组合(共 **54%**,略低于 60%):
- 主消 12% / 医药 12% / 信息 12% / 电信 6% / 金融 6% / 能源 6%
- + 类现金 46%

### 6.5 调整节奏

不是一次性切换——**分 3 步,跨季度执行**:

| 日期 | 动作 |
| --- | --- |
| 2011-01-15 | 公用 12% → 6%(减半),释放 6% 到类现金 |
| 2011-02-15 | 公用 6% → 0%,引入金融 3% + 能源 3% |
| 2011-03-15 | 电信 12% → 6%,补金融 3% + 能源 3% |

3 月 15 日完成,**总仓位 54%**。

### 6.6 与 2009 年 / 2010 年决策的对比

| 时点 | 状态指标 | 股仓位 | 主要思路 |
| --- | ---:| ---:| --- |
| 2008.01 | 78% | 30% | 极端危险,大幅降仓 |
| 2009.01 | 22% | 75% | 极端机会,大幅加仓 |
| 2010.01 | 50% | 60% | 中性,从 75% 减到 60% |
| **2011.01** | **46%** | **54%** | **中性偏紧,从 60% 微降到 54%** |

**避极端框架的一致性体现**:小变化不大动,只在状态显著变化时调整。

## 七、加仓/降仓信号清单(进入 2011 后实时观察)

### 7.1 加仓信号(任一触发 +5%)

| 信号 | 阈值 |
| --- | --- |
| 沪深300 PE-TTM 跌破 13 | 极端便宜 |
| CPI 连续 3 个月 < 3% | 通胀压力解除 |
| 央行停止加息(连续 4 个月不加) | 政策转向 |
| 国务院"稳增长"基调出现 | 政策面利好 |
| 创业板 / 中小板跌幅 > 30% | 次生泡沫破裂 |

### 7.2 降仓信号(任一触发 -5%)

| 信号 | 阈值 |
| --- | --- |
| CPI 连续 2 个月 > 6% | 通胀失控 |
| 央行加息节奏加快(单年 ≥ 4 次) | 硬紧缩 |
| 欧债扩散到西班牙 / 意大利 | 系统性外部冲击 |
| 沪深300 反弹到 PE-TTM > 20 | 估值过热 |
| 房地产销售连续 6 个月负增长 | 经济失速 |

### 7.3 不做信号的事

- 不预测 CPI 哪个月见顶——观察后跟随
- 不预测欧债是否传染——观察传染指标
- 不预测大盘是否突破 3500——按估值定位

## 九、一句话总结

> **2011 年初,理性人面对的是一个"不极端,但很难"的市场**。
>
> 没有 2008 那种"政策全面救市 + 估值 12x"的明确机会,
> 也没有 2009.08 那种"PE 32x + 通胀传导"的明确危险。
>
> **真正的考验是:在 46% 的中性偏紧状态下,坚持不被"小盘股牛市"诱惑、不被"PE 15 很便宜"诱惑、不被"通胀见顶"诱惑**。
>
> 决策本身很简单:**54% 仓位 + 6 个均衡行业(去掉公用,加入金融+能源)**。
>
> 难的是**接下来 12 个月按这个不动**——
> **如果 2011 真的演变为"小型 2008",才会触发系统性加仓;
> 否则就守住 54%,赢在不犯错**。
