## 三、关键决策复盘

### 决策 1: 2009.06 → 7 月先减一档 (80% → 75%)

- **触发**: 评分从 -6 跳到 -4(估值升 PE→25,M1 大升)
- **结果**: 7 月仍享受 +17.94% 涨幅,但暴露下降
- **评价**: 中性,主要为下一步铺垫

### 决策 2: 2009.07 → 8 月**强减一档**(75% → 60%)⭐ 核心

- **触发**: A=2.0 + B=1.5 + C=1.5 + D=2.0 = **倍数 1.75 全维放大**
  - A: 央行 7.23 报告首次"动态微调" → 宽松末期警报
  - B: M1=27 + 信贷天量 → 信贷溢出资金主导
  - C: 牛市仅 9 个月 → 短反弹动能薄
  - D: PE 同比 +70% / EPS 同比 +5% → 严重透支
- **额外档**: extra_step=-1(75%→60%),因 A=2.0 + base ≤ -3 + 过热信号存在
- **结果**: 8 月 -24.22% 暴跌,**v3.6 仅亏 -14.5%(60% 仓位)vs v3.4 亏 -21.8%(90% 仓位)**
- **评价**: ⭐ **决定性胜利**,避开了主跌的 24 个百分点敞口

### 决策 3: 2009.08 暴跌后 → 9 月加回 75%

- **触发**: 评分仍是 -2.2(动能转换 + price-in 反吐压制)
- **关键**: v3.4 此时评分 -11 → 90% 仓位(被基本面强 + 情绪冷迷惑)
- **v3.6 优势**: 不被"看似低估"诱惑,因为 A=2.0 仍生效,知道 7 月已是中期顶部
- **结果**: 仓位 75% 而非 90%,后续 9-12 月反弹温和(每月 +2~+9%)收益略低,但避免了 V 字形二次崩盘的风险

### 决策 4: 2009.09 → 10-12 月稳定 80%

- A 维度退出(无新"微调"信号)→ 回归正常评分逻辑
- 评分回到 -7~-9 强多头区
- 仓位维持 80%,顺利收割反弹

## 五、本质洞察 — 为什么 v3.6 成功

**核心机制**: 当**4 维动能全部放大**(2009.07)时,v3.6 启动两层防护:

```
第一层: 过热扣分 × 1.75 倍 → 评分提升 4.8 分
第二层: 利多 price-in 反吐 80% → 评分再提升 4.8 分
第三层: extra_step 仓位档强制 -1 → 75% → 60%

最终: v3.4 评分 -6（看多）→ v3.6 评分 -2.7 + 额外减档（防守）
```

**普世原则验证**: 用户洞见 **"外部信号必须经过市场内部动能的转换才能定价"** 在 2009.07 得到完美体现——同样是 PE>30、新基爆款、两融激增,2007.05 被市场吸收(继续涨 5 月),2009.07 被市场放大(次月崩 24%),**差别在 4 个结构性条件**。
