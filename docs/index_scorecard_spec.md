# 指数评分卡评分标准（Index Scorecard）

**版本**：v2.0（2026-06）
**定位**：V5.0 宏观评分卡的第二层，解决"买哪个指数"的权重分配问题
**前置依赖**：`calc_index_pe_pb.py` 的 PE/PB 历史数据 + `index_daily` 价格数据

---

## 一、核心思想

V5.0 评分卡解决**"买多少"**（总仓位 0~100%）；指数评分卡解决**"买哪个"**（已定仓位下各指数的相对权重）。

```
Layer 1  V5.0 宏观评分卡（已有）
         ↓ 总仓位 target_equity（0~100%）
Layer 2  指数评分卡（本文）
         ↓ 各指数相对权重 weight_i（Σ weight_i = 1）
最终仓位  position_i = target_equity × weight_i
```

**两层完全独立**：进入指数评分卡时，总仓位已由 V5.0 决定，这里只做跨指数的相对排序。宏观信号（PMI、利率、美联储等）不在本层重复计算；若后续回测发现牛熊市中资金偏好有显著差异，再以独立维度补入。

### 红线

1. **不影响总仓位** — 指数评分卡只影响结构分配
2. **估值阈值用历史分位数** — 不同指数 PE 量级差异巨大（沪深300 ≈ 14，创业板 ≈ 55），绝对阈值无意义
3. **缺失数据不计分** — 历史不足时返回中性（0 分），不编造信号

---

## 二、四大维度

| 维度 | 评分区间 | 数据来源 |
|:-----|:--------:|:---------|
| A. 估值分位（Valuation Percentile） | [-4, +4] | PE/PB 历史分位 |
| B. 盈利质量（Earnings Quality） | [-2, +2] | 隐含 ROE 斜率（PB/PE×100） |
| C. 价格动量（Price Momentum） | [-3, +3] | index_daily 三时间维度 |
| D. 结构偏离（Structural Deviation） | [-1, +1] | 价格距 3 年高低点距离 |

**总分范围**：[-10, +10]

---

## 三、维度详细规则

### A. 估值分位（valuation_percentile）

**评分区间**：[-4, +4]

| 规则 | 条件 | 方向 | 评分 |
|:-----|:-----|:----:|:----:|
| PE 历史极低位 | `pe_pctl_10y < 15` | opportunity | **−2** |
| PE 历史偏低 | `15 ≤ pe_pctl_10y < 30` | opportunity | −1 |
| PE 历史偏高 | `70 < pe_pctl_10y ≤ 85` | risk | +1 |
| PE 历史极高位 | `pe_pctl_10y > 85` | risk | **+2** |
| PB 历史极低位 | `pb_pctl_10y < 15` | opportunity | −1 |
| PB 历史极高位 | `pb_pctl_10y > 85` | risk | +1 |
| 股息率极高位 | `dv_pctl_10y > 80`（仅当指数平均股息率 > 1.5%） | opportunity | −1 |

**变量说明**：
- `pe_pctl_10y`：当前 PE_TTM 在自身过去 10 年历史中的分位数（0~100）
- `pb_pctl_10y`：当前 PB 在自身过去 10 年历史中的分位数（0~100）
- `dv_pctl_10y`：当前股息率在自身过去 10 年历史中的分位数

**为什么 PE 和 PB 分开计分**：两者可以背离——PE 低但 PB 高说明当期盈利好但资产质量溢价已高（周期行业顶部常见）；PE 和 PB 双低才是更可信的底部信号。股息率高位作为红利类指数的额外加仓信号，成长类指数股息率长期为 0，该规则自然不触发。

**与 V5.0 估值的关系**：V5.0 用沪深300的绝对 PE（>50/40/<15）判断总仓位；指数层用各指数**自身**历史分位数做相对排序，两者不冲突。

---

### B. 盈利质量（earnings_quality）

**评分区间**：[-2, +2]

**核心变量**：`implied_roe = index_pb / index_pe_ttm × 100`（指数加权隐含 ROE %）

| 规则 | 条件 | 方向 | 评分 |
|:-----|:-----|:----:|:----:|
| 隐含 ROE 持续提升 | `roe_3y_slope > +1.5 pp/年` 且方向一致 | opportunity | **−2** |
| 隐含 ROE 轻度改善 | `0.5 < roe_3y_slope ≤ 1.5 pp/年` | opportunity | −1 |
| 隐含 ROE 轻度恶化 | `-1.5 ≤ roe_3y_slope < -0.5 pp/年` | risk | +1 |
| 隐含 ROE 持续下滑 | `roe_3y_slope < -1.5 pp/年` 且方向一致 | risk | **+2** |

**变量说明**：
- `roe_3y_slope`：以过去 3 年年末 `implied_roe` 做线性回归，单位 pp/年
- "方向一致"：3 年中至少 2 年单调同向变化

**为什么需要这个维度**：估值便宜但盈利持续恶化是典型的价值陷阱（PE 低是因为 EPS 在萎缩）。盈利质量维度在估值维度的基础上做方向验证：PE 分位低 + ROE 斜率为正 = 真底部；PE 分位低 + ROE 斜率持续为负 = 估值陷阱，两者综合后估值虚低的信号被盈利质量部分对冲。

**数据来源**：`implied_roe` 从 `index_dailybasic` 的 PE/PB 历史直接推导，无需额外数据表。

---

### C. 价格动量（price_momentum）

**评分区间**：[-3, +3]

| 维度 | 规则 | 条件 | 方向 | 评分 |
|:-----|:-----|:-----|:----:|:----:|
| 12M 中期 | 深度超跌 | `index_12m_return < -30%` | opportunity | −1 |
| 12M 中期 | 极度过热 | `index_12m_return > +50%` | risk | +1 |
| 6M 相对强度 | 行业领先 | 6M 超额收益在指数池内排名前 25% | opportunity | −1 |
| 6M 相对强度 | 行业落后 | 6M 超额收益在指数池内排名后 25% | risk | +1 |
| 1M 短期 | 极度超卖 | `index_1m_return < -8%` | opportunity | −1 |
| 1M 短期 | 极度超买 | `index_1m_return > +10%` | risk | +1 |

**变量说明**：
- `index_12m_return`：近 12 个月累计涨跌幅（%）
- `index_1m_return`：近 1 个月涨跌幅（%）
- `超额收益排名`：各指数 6M 涨跌幅 vs 沪深300 同期，在指数池内排序取分位

**三层动量的逻辑**：
- **12M 绝对动量**：中期趋势的极端状态。年跌 30% 以上通常是出清信号（均值回归机会），年涨 50% 以上是泡沫风险
- **6M 相对强度**：截面动量，行业轮动的延续性信号。跑赢沪深300 的行业倾向于继续领先
- **1M 短期反转**：月内极端波动往往是非理性恐慌或亢奋，-8% 以下超卖提供短期买点，+10% 以上超买是减仓窗口

---

### D. 结构偏离（structural_deviation）

**评分区间**：[-1, +1]

| 规则 | 条件 | 方向 | 评分 |
|:-----|:-----|:----:|:----:|
| 贴近 3 年低点 | `price_vs_3y_low_pct < 5%` | opportunity | −1 |
| 贴近 3 年高点 | `price_vs_3y_high_pct < 5%` | risk | +1 |

**变量说明**：
- `price_vs_3y_low_pct`：当前价格比过去 3 年最低价高出的百分比（< 5% 即"贴近低点"）
- `price_vs_3y_high_pct`：过去 3 年最高价比当前价格高出的百分比（< 5% 即"贴近高点"）

这是最简单的技术结构信号：价格接近 3 年低点通常意味着阶段性恐慌或供给出清，接近 3 年高点则有明确压力位。作为辅助确认信号，单独不足以决策，但可以修正估值和动量的边界判断。

---

## 四、总分与权重映射

**总分计算**：

```
index_score = A.估值分位 + B.盈利质量 + C.价格动量 + D.结构偏离
```

**理论区间**：[-10, +10]

**指数评分 → 相对权重调整系数**（基准权重 = 等权或市值加权）：

| 指数评分 | 调整系数 | 含义 |
|:--------:|:--------:|:-----|
| `score ≤ -6` | **2.0×** | 极度低估+盈利改善，显著超配 |
| `-5 ~ -4` | **1.5×** | 明显机会，超配 |
| `-3 ~ -2` | **1.2×** | 轻度低估，小幅超配 |
| `-1 ~ +1` | **1.0×** | 中性，维持基准 |
| `+2 ~ +3` | **0.7×** | 轻度高估，小幅低配 |
| `+4 ~ +5` | **0.5×** | 明显高估，低配 |
| `score ≥ +6` | **0.2×** | 极度高估，显著低配 |

**归一化**：

```python
raw_weight_i   = base_weight_i × adjustment_i
final_weight_i = raw_weight_i / sum(raw_weight_j for all j)
```

**约束**：
- 单只指数权重上限：30%（避免极端集中）
- 单只指数权重下限：2%（避免完全清零，降低再平衡成本）

## 五、数据需求与来源

| 字段 | 数据表 | 取数口径 |
|:-----|:-------|:---------|
| `index_pe_ttm` | `index_dailybasic` | 成份股加权 PE_TTM（`calc_index_pe_pb.py` 计算） |
| `index_pb` | `index_dailybasic` | 成份股加权 PB |
| `pe_pctl_10y` | 运行时计算 | 当前 PE 在自身 10 年历史中的分位数（0~100） |
| `pb_pctl_10y` | 运行时计算 | 当前 PB 在自身 10 年历史中的分位数 |
| `dv_pctl_10y` | 运行时计算 | 当前股息率在自身 10 年历史中的分位数 |
| `implied_roe` | 运行时计算 | `index_pb / index_pe_ttm × 100` |
| `roe_3y_slope` | 运行时计算 | 过去 3 年年末 `implied_roe` 线性回归斜率（pp/年） |
| `index_12m_return` | `index_daily` | 12 月前至今累计涨跌幅（%） |
| `index_1m_return` | `index_daily` | 1 月前至今涨跌幅（%） |
| `rel_strength_rank_6m` | 运行时计算 | 各指数 6M 超额收益在指数池内的排名分位 |
| `price_vs_3y_low_pct` | `index_daily` | 当前价格比 3 年最低点高出的百分比 |
| `price_vs_3y_high_pct` | `index_daily` | 3 年最高点比当前价格高出的百分比 |

**历史数据深度要求**：
- 分位数计算（维度 A）：至少 5 年 PE/PB 历史，不足时跳过
- ROE 斜率（维度 B）：至少 3 年年末数据，不足时跳过
- 12M 动量（维度 C）：至少 12 个月价格历史，不足时跳过该规则

## 六、IndexScorecardInputs 字段定义

| 字段 | 类型 | 说明 |
|:-----|:----:|:-----|
| `index_code` | str | 指数代码（如 `931233.CSI`） |
| `pe_pctl_10y` | float \| None | PE 10 年历史分位数（0~100） |
| `pb_pctl_10y` | float \| None | PB 10 年历史分位数（0~100） |
| `dv_pctl_10y` | float \| None | 股息率 10 年历史分位数（0~100，仅红利类有效） |
| `index_avg_dv` | float \| None | 当前指数平均股息率（%，用于判断是否触发股息率规则） |
| `roe_3y_slope` | float \| None | 隐含 ROE 3 年线性斜率（pp/年） |
| `roe_3y_monotone` | bool \| None | 3 年中至少 2 年方向一致 |
| `index_12m_return` | float \| None | 近 12 月累计涨跌幅（%） |
| `index_1m_return` | float \| None | 近 1 月涨跌幅（%） |
| `rel_strength_rank_6m` | float \| None | 6M 超额收益池内排名分位（0~100） |
| `price_vs_3y_low_pct` | float \| None | 当前价格距 3 年低点的百分比（%） |
| `price_vs_3y_high_pct` | float \| None | 3 年高点距当前价格的百分比（%） |

所有字段为 None 时跳过对应规则，不影响其他维度得分。

## 八、设计决策记录

| 决策点 | 选择 | 理由 |
|:-------|:-----|:-----|
| 是否纳入 V5.0 宏观信号 | **不纳入** | 宏观信号已在总仓位层处理完毕；指数层只做相对排序，无需重算 |
| 估值：PE vs PE+PB | **PE+PB 各自独立** | 两者可背离，双信号识别估值陷阱能力更强 |
| 是否加盈利质量维度 | **加** | 防止 PE 低但 ROE 持续下滑的价值陷阱，是估值维度的必要补充 |
| 动量时间维度 | **12M + 6M相对 + 1M** | 12M 捕捉中期趋势；6M 截面动量有延续性；1M 短期反转辅助择时 |
| 结构偏离 | **3 年高低点距离** | 简单有效的技术支撑/压力信号，无需新数据，从 index_daily 直接计算 |
| 是否加指数类型分类 | **暂不加** | 首版保持简单；宏观共振待回测验证后再以独立维度补入 |
