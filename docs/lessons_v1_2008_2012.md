# 上一轮推演 (2008-2012) 的教训沉淀
> 这一轮虽然战绩好（5 年累计 +24.74%，跑赢沪深300 56.7pp），但用户判定**"失败"**——失败在框架，不在数字

## 一、表面上学到的（"看似"理性人能力）

| 能力 | 来源 | 是否普适 |
|---|---|---|
| 极端低估（PE<10、PB<1.4）触发加仓 | 2008.11、2012.12 都识别到了 | ✅ 是 |
| 高估（PE>30、股息率<2%）触发减仓 | 2009.11、2010.11 减仓避开 2011 | ✅ 是 |
| 仓位上下限（30%-85%）替代择时 | 不空仓也不满仓 | ✅ 是 |
| 类现金 / 中长期债做缓冲层 | 平滑回撤 | ✅ 是 |
| 预案触发机制（PE/降息/CPI/股息率） | 把决策变规则 | ✅ 是 |
| ~~用上证红利 510880 做"防御核心"~~ | 上一轮的特定方案 | ❌ **不是普适经验** |
| ~~把"行业仓位"做主动调整~~ | 2012 加金融到 16% 跑赢 | ❌ **被用户证伪的路径依赖** |

## 二、真正学到的（用户深度质疑后暴露的盲区）

### 盲区 1: "行业框架"是路径依赖
**用户原话**："按行业来设计自己的资产组合，大概率是会很难抓住市场机会的，比如 AI 其实没有行业，新能源也没有行业..."

→ 行业 ETF（金融/医药/消费/军工）是 2006 起的旧框架，对**主题驱动的成长股盲区**。
→ 真正驱动 2013+ 收益的是：移动互联网、新能源车产业链、AI 算力、机器人、自由现金流策略。
→ 行业划分是 GICS 给的，**市场涨的不是行业是主题/产业链**。

### 盲区 2: "避免极端"被错误执行成"追求最优"
**用户原话修正**："我说的避免极端是说，不需要考虑一定找到收益最高的方式，只要能保证我当下的选择，可以有极大概率避免踩中极端的风险。"

→ 上一轮做错了：把"避免极端"当成"找到最优配置比例"
→ 正确做法：**判断不是"该买什么"，而是"哪些情况一定不能踩"**

### 盲区 3: "预案叠加"会过度敏感
2012 年首次跑输基准 (-3.37pp) 的根本原因：
- 估值预案没触发 (PE 9.54 还未到<10)
- 但**降息预案 + CPI 预案叠加**把仓位从 74% 推到 89%
- 加仓后被磨 -10%

→ 学到：**预案应该是"地板"而不是"加速器"**。多个预案同时触发，取最严的一条，不累加。

### 盲区 4: "可用工具" 没有严格按发布日校准
上一轮在 2008/2009 错误地默认了"沪深300 ETF 可用" (实际 2012.05.28 才上市)、"创业板 ETF 可用" (2011.09 才上市)、"中证红利 ETF 可用" (2019.12 才上市)
→ 这一轮**必须严格按 ETF 实际上市日**给工具池

## 三、本轮 (2006-2026) 升级后的"理性人"六原则

### 原则 0（最高原则）：**评价标准是长周期高收益 + 穿越极端**
- 不是"20 年没踩极端"——只防御不进攻没有意义
- 不是"每年跑赢基准"——单年跑输可以接受
- 是 **20 年累计 CAGR 跑赢沪深300 全收益 + 任何 3 年滚动期最大回撤可控**
- 进攻和防御都要做，**关键是不在错的时点做错的事**

### 原则 1：**工具池硬约束**
- 当年初决策只能用**已上市的 ETF**
- 不存在的 ETF 不进决策，即使指数早已发布
- 例如：2006.01.04 真正可用的 A 股 ETF 只有 510050 一只
- 工具不足时用**主动基金 / LOF / 货基 / 国债**过渡，不强行用不存在的工具

### 原则 2：**避免极端 ≠ 寻找最优**
判定路径变为：
1. 列出"踩中重大损失"的所有具体情景（包括**错失重大上涨**也算损失）
2. 对每个情景，设计"不会踩中"的最小约束
3. 在满足约束的所有方案中，选**预期收益最高**的那个（不是"最简单"）
4. 绝不为了 +2% 的预期收益增加 2 个变量

### 原则 3：**框架不固化**
每 3 年强制问自己：
- 当前框架的核心驱动假设还成立吗？
- 这 3 年市场涨的钱，我的框架解释得了吗？
- 如果一个新维度 (主题/风格/跨境/商品) 涨幅最大，我有触角吗？
- **该不该把组合从"宽基+行业"换成"宽基+主题"？从"A股"扩到"A+港+美+商品"？**

### 原则 4：**估值是地板和天花板，不是加速器**
- 极端低估 (PE<10) → 股票仓位**下限**抬高到 75%（不被动减仓，进攻地板）
- 极端高估 (PE>30) → 股票仓位**上限**压到 50%（不追加，防御天花板）
- 中性区间 (PE 12-25) → 看预案，但单一预案不超过 ±3% 仓位调整
- **任何叠加预案都不能突破 85% 总仓位上限**（v1 是 89% 出过事）

### 原则 5（新）：**不预设核心资产**
- 不预先指定"红利做核心" / "沪深300做核心" / "成长做核心"
- 核心资产由**每年的估值 + 工具池 + 宏观**重新选
- 任何资产仓位 > 30% 都要重新论证
- 路径依赖（"上一年这么配所以今年也这么配"）是禁止的

## 四、本轮的关键调整

| 维度 | 上一轮 (v1) | 本轮 (v2) |
|---|---|---|
| 起点 | 2008 年初 | **2006 年初** |
| 工具池 | 默认所有指数可用 | **严格按 ETF 上市日** |
| 决策框架 | 行业仓位为主 | **宽基/风格/主题/跨境/商品/债 多维并行，每年重选核心** |
| 仓位上限 | 89% (出过事) | **85%** |
| 预案叠加 | 累加加仓 | **取最严的那条做地板** |
| 核心资产 | 红利做防御核心 | **不预设核心，每年重选** |
| 评价标准 | 跑赢基准 = 成功 | **20 年 CAGR 高 + 3 年滚动回撤可控 = 成功** |

# 2006 年初开局：理性人看到了什么

## 时间锚点：2006 年 1 月 4 日（首个交易日）

理性人此刻只能看到 **2005.12.30 之前的所有公开信息**。

## 一、宏观环境（理性人 2005 年底的认知）

### 1. 中国宏观（2005 全年）
- GDP 增速：**10.4%**（2003 起连续第三年 10%+，但 1998-2002 经历过 7-9%）
- CPI：**1.8%**（温和，国务院刚启动稳健货币政策）
- 1 年期定存利率：**2.25%**（2004.10.29 加息后维持）
- 房改启动 7 年，城镇化加速；2005 房地产投资增速 +19.8%
- 关键事件：**股权分置改革 2005.04.29 启动**（中国证券市场最重大制度变革）

### 2. 全球宏观
- 美联储 2004.06 启动加息周期，2005 末联邦基金利率从 1.0% → 4.25%
- 油价 2005 年内冲到 $70+（卡特里娜飓风）
- 美元强势，新兴市场资金流出

### 3. 国内股市（理性人能看到的 5 年熊市）
- 上证综指 2001.06.13 见顶 **2242** → 2005.07.11 见底 **1011** = **-54.9%**（5 年熊市）
- 上证50 三年累计 -21.3%，最大回撤 -38.3%
- 2005 全年小幅反弹但仍在底部区域震荡

## 四、2006 年初的资产配置决策

### 决策原则（按本轮升级后的六原则）

**原则 0（长周期高收益）**：现在 PE 13.78 是 5 年熊市末端，**进攻概率 > 防御概率**
**原则 1（工具池硬约束）**：只能用 510050 + 主动基金 + 货基 + 国债
**原则 4（估值是地板）**：PE 13.78 < 15，触发"极端低估地板"→ 股票仓位**下限 75%**
**原则 5（不预设核心）**：不假设"红利做核心"，因为 510880 11 月才上市；初始核心由可得性决定

**原则 2（避免极端）的双向情景**：

| 情景 | 损失类型 | 概率 |
|---|---|---|
| A. 5 年熊市第 6 年继续杀 -30% | 资本永久损失 | 低（估值已极低） |
| B. 股权分置改革大小非解禁砸盘 | 短期波动 | 中（2007-2008 高峰） |
| C. 全球加息引发资金外流 | 短期波动 | 中 |
| **D. 错过底部回升** | **机会成本永久损失** | **高（最大极端）** |

→ 在 PE 13.78 的低估区，**D 远比 A/B/C 严重**。理性人应该**进攻**，而不是因为"5 年熊市"的心理阴影而保守。

### 仓位决策（修正：75% 股票地板）

| 资产 | 仓位 | 工具 | 理由 |
|---|---:|---|---|
| **大盘 ETF** | **35%** | 510050 上证50ETF | 唯一 ETF，估值最便宜 (PE 13.78) |
| **主动股票基金** | **40%** | 华夏大盘精选 25% + 富国天益 15% | 工具池不足时的必要补充，主动选股 |
| **国债 / 类债** | **15%** | 5 年期记账式国债 + 国债逆回购 | 全球加息减少久期 |
| **货基** | **10%** | 华安现金 / 博时增长 | 等待 11 月 510880 / 用于再平衡 |
| 合计 | **100%** | | 股票 **75%**（命中地板）|

**为什么是 75% 不是 85%**：
- 估值低 → 进攻，地板 75%
- 但**主动基金有人为风险**（基金经理变更、风格漂移）→ 不能用主动基金推到 85%
- 一旦 11 月 510880 上市，可以用 ETF 替换部分主动基金，进一步提高确定性

### 全年预案

| 触发条件 | 动作 |
|---|---|
| **2006.04.24** 510180 + 159901 上市 | 主动基金 → 510180 (10%) + 159901 (5%)；首次拿到深市 beta |
| **2006.06.30** 159902 上市 | 视情况加 5% 中小板（首次拿到小盘 beta） |
| **2006.11.17** 510880 上市 | 主动基金 → 510880 (10%)；**首个风格因子 ETF**，但不预设为"核心" |
| 沪深300 PE 上行至 20-25 | 维持仓位，不减仓 |
| 沪深300 PE > 25 | 减仓 5%，从主动基金开始减 |
| 沪深300 PE > 30 | 减仓到股票 60% |
| 沪深300 PE > 40 | 减仓到股票 50%（极端高估天花板）|
| 央行加息 ≥ 2 次 | 缩短国债久期，不动股票 |
| 全年回撤 > -20% | **不减仓**（地板原则）|

## 五、与上一轮的关键差异

1. **起点更早 2 年**（2006 vs 2008）—— 多走过股权分置改革 + 2007 大牛市 + 2008 危机这三个关键场景
2. **工具池更窄**（只有 510050 vs 上轮误以为有沪深300）—— 用主动基金过渡，但不依赖
3. **仓位更高**（75% vs 上轮初始 60%）—— 估值 PE 13.78 应该进攻
4. **不预设核心**（vs 上轮把红利当核心）—— 核心由估值 + 工具池逐年重选
5. **不预测，不抢跑**—— 不预设 2007 牛市来，不预设 2008 危机来
