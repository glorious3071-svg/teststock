# 反思：评分卡的单向化缺陷与双向化重构

## 一、你抓住的根本问题

v3.3b 评分卡是**单向风险评分**：
- 每个维度只有"风险加分"
- 没有"机会减分"

这导致：
- **市场过热时（2007.09）**：评分=16，正确指示减仓 ✅
- **市场极度恶化但估值已极便宜（2009.01）**：评分=8（基本面+外部仍计 8 分），错失加仓信号 ❌

**根本错位**：评分卡的核心问题不是"基本面差不差"，而是"**风险vs机会的相对位置**"。基本面差但估值已打折，相当于"利空出尽"——本身是**机会**而非**风险**。

---

## 二、单向 vs 双向的本质差异

### 单向评分（v3.2 / v3.3b）

```
评分 = Σ 风险信号 (≥0)
风险高 → 减仓
```

**问题**：底部时所有维度"看起来都坏"，但坏消息已经定价。

### 双向评分（v3.4）

```
评分 = Σ (风险信号 - 机会信号)
评分高 → 风险占优 → 减仓
评分低 → 机会占优 → 加仓
```

**关键**：每个维度都有"加分"和"减分"两个方向，**净评分**才是决策依据。

## 六、v3.4 关键意义

### 1. 解决了 v3.3b 的"底部失灵"

| 时点 | v3.3b 评分 | v3.4 评分 | v3.4 改进 |
|---|:-:|:-:|---|
| 2009.01 | 8（50%仓位） | **−5（80% 仓位）** | **彻底识别底部** |
| 2007.09 顶部 | 16（20%仓位） | 16-17（20%仓位） | 顶部识别不变 |
| 2008.10 暴跌中 | 9（50% 但v3.3b放行 35%） | −3（75%） | 估值+流动性已反转，机会上升 |

### 2. 引入"反身性"——好消息也是信号

巴菲特：**别人恐慌时贪婪**。但单向评分会让你在"基本面差"时减仓，**永远等不到大底**。

v3.4 的双向机制把"估值便宜""政策宽松""美联储零利率"这些**正向信号**也计入评分，让你能在底部识别机会。

### 3. 保留了 v3.3b 的好东西

- 减仓时立即响应（高分 → 立刻减）
- 加仓时需通过"政策实弹三重门"（防止过早抄底）
- 估值兜底（PE<15+PB<2 自带 −3 分，几乎自动触发加仓档）

需要用 v3.3b 的"加仓三重门"约束 v3.4 的过早加仓——这两个机制是**互补**的。

## 八、关于"对称性"的反思

### 一个常见误区：单向评分更"严格"

很多人觉得单向风险评分"更保守、更稳健"。但 v3.3b 在 2009 的失灵告诉我们：

**只看风险=只能赚下跌的钱（减仓），赚不到上涨的钱（加仓）**

真正稳健的框架应该**双向对称**：
- 风险信号 → 减仓
- 机会信号 → 加仓
- 净信号决定方向

### 但要警惕"机会信号过度敏感"

如果机会信号触发太容易，会变成"每次小利好就加仓"——和 v3 严格纪律失败的原因相反但同样糟糕。

v3.4 的机会信号设计原则：
- **必须是历史极端**：PE<15 不是 PE<25
- **必须是政策实弹**：央行已降息 100bp，不是央行表态
- **必须是结构性变化**：美联储零利率+QE，不是单次降息

## 九、对 v3.4 的剩余疑虑

### 疑虑 1：2008.10 提前加仓的风险

v3.4 评分 −3 → 75% 仓位，但 10 月本月仍跌 -26%。这部分跌幅会被 75% 仓位放大。

**解法**：v3.3b 的"加仓三重门"仍然适用——本月跌 >15% 暂缓加仓。v3.4 评分给出"目标 75%"，但实际执行受三重门约束。

### 疑虑 2：双向机制是否过拟合 2009

我们设计的"机会信号"完全是从 2009 的特征反推的：PE<15、累计降息 >100bp、美联储零利率... 这些都是**用未来视角设计**的吗？

**核查**：
- PE<15 是历史 10% 分位以下，**任何时代都适用**
- 累计降息 >100bp 是宽松周期客观指标
- 美联储零利率是 2008 后才出现的"新常态"

这些不是 2009 专属，**未来 2015/2020 同样适用**。

### 疑虑 3：评分范围扩大到 [−15, +25] 是否太敏感

档位映射改了：
- 原 v3.3b：[0, 25] → 5 档
- 新 v3.4：[−15, +25] → 8 档

**意义**：让仓位变化更细腻（90% / 80% / 75% / 60% / 50% / 30% / 20%），不是单纯增加敏感性。
